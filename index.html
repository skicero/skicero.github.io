Présentation du projet :

Le projet que nous avons choisi de faire est une application mobile de vente de vêtements d'une start-up dont la marque est RAR.
Nous avons trouvé l'idée de cette application car les deux patrons de cette start-up sont des copains que je connais depuis la Seconde.
Cette opportunitée nous à pas fallu longtemps pour la prendre car dans les deux sens nous étions content : la start-up avait alors une application mobile faite gratuitement et nous de notre côté elle pouvait nous servir pour réaliser notre projet.
